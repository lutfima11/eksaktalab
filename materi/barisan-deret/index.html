APengertian Barisan dan Deret

1. Apa Itu Barisan Bilangan?

Perhatikan susunan bilangan berikut:

$$2,\quad 4,\quad 6,\quad 8,\quad 10,\quad \ldots$$

Tampak bahwa setiap bilangan berikutnya bertambah 2. Ada pola yang jelas dan teratur di sini.

Sekarang perhatikan contoh lain:

$$3,\quad 9,\quad 27,\quad 81,\quad 243,\quad \ldots$$

Setiap bilangan diperoleh dengan mengalikan bilangan sebelumnya dengan 3. Ini juga sebuah pola yang teratur.

Kedua contoh di atas adalah contoh barisan bilangan — susunan bilangan yang mengikuti aturan (pola) tertentu. Setiap bilangan dalam barisan disebut suku. Suku pertama ditulis $U_1$, suku kedua $U_2$, suku ketiga $U_3$, dan seterusnya. Suku ke-$n$ ditulis $U_n$.

Contoh barisan dan suku-sukunya:

Barisan$U_1$$U_2$$U_3$$U_4$$U_5$Pola
$1,\; 3,\; 5,\; 7,\; 9,\; \ldots$13579ditambah 2 setiap suku
$2,\; 6,\; 18,\; 54,\; \ldots$261854162dikali 3 setiap suku
$1,\; 4,\; 9,\; 16,\; 25,\; \ldots$1491625bilangan kuadrat

2. Apa Itu Deret Bilangan?

Jika suku-suku suatu barisan dijumlahkan, hasilnya disebut deret bilangan. Misalnya:

Barisan:$2,\quad 4,\quad 6,\quad 8,\quad 10$
Deretnya:$2 + 4 + 6 + 8 + 10 = 30$

Jumlah $n$ suku pertama dari suatu deret dilambangkan dengan $S_n$. Sehingga:

$$S_n = U_1 + U_2 + U_3 + \cdots + U_n$$
⚠ Perbedaan Penting

Barisan $\to$ suku-suku dipisahkan tanda koma: $\;2,\; 5,\; 8,\; 11,\; \ldots$
Deret $\to$ suku-suku dijumlahkan: $\;2 + 5 + 8 + 11 + \cdots$

3. Hubungan Suku ke-$n$ dan Jumlah $n$ Suku

Dari definisi $S_n = U_1 + U_2 + \cdots + U_{n-1} + U_n$, kita dapat melihat bahwa:

$$S_{n-1} = U_1 + U_2 + \cdots + U_{n-1}$$

Sehingga $U_n = S_n - S_{n-1}$. Hubungan ini sangat berguna ketika rumus $S_n$ diketahui tetapi rumus $U_n$ tidak diketahui secara langsung.

• Hubungan $U_n$ dan $S_n$
$$U_1 = S_1 \qquad \text{dan} \qquad U_n = S_n - S_{n-1} \quad \text{untuk } n \geq 2$$
📘 Contoh 1

Diketahui $S_n = 3n^2 + n$. Tentukan $U_1$ dan $U_3$!

Penyelesaian:
  1. $U_1 = S_1 = 3(1)^2 + 1 = 4$
  2. $S_3 = 3(9) + 3 = 30$ dan $S_2 = 3(4) + 2 = 14$, sehingga $U_3 = S_3 - S_2 = 30 - 14 = 16$
📘 Contoh 2

Diketahui $S_n = an^2 + bn$. Jika $U_3 = 11$ dan $U_5 = 19$, tentukan nilai $a$, $b$, dan $S_{10}$!

Penyelesaian:
Karena $U_n = S_n - S_{n-1} = an^2 + bn - a(n-1)^2 - b(n-1) = 2an - a + b$, maka: $$U_3 = 5a + b = 11 \quad \cdots(1)$$ $$U_5 = 9a + b = 19 \quad \cdots(2)$$ Dari $(2) - (1)$: $4a = 8 \Rightarrow a = 2$. Substitusi ke (1): $b = 1$. $$S_{10} = 2(100) + 1(10) = 210$$
📘 Contoh 3

Diketahui jumlah $n$ suku pertama suatu barisan dinyatakan oleh $S_n = 2n^2 + n$. Tentukan nilai $U_1$, $U_2$, $U_3$, dan $U_n$ untuk $n \geq 2$!

Penyelesaian:
Untuk $U_1$, gunakan $U_1 = S_1$: $$U_1 = S_1 = 2(1)^2 + 1 = 3$$ Untuk $n \geq 2$, gunakan $U_n = S_n - S_{n-1}$: $$S_n = 2n^2 + n$$ $$S_{n-1} = 2(n-1)^2 + (n-1) = 2n^2 - 4n + 2 + n - 1 = 2n^2 - 3n + 1$$ $$U_n = S_n - S_{n-1} = (2n^2 + n) - (2n^2 - 3n + 1) = 4n - 1$$ Cek kecocokan dengan $U_1$: $4(1)-1 = 3$ ✓

Nilai suku-suku awal: $$U_2 = 4(2)-1 = 7, \qquad U_3 = 4(3)-1 = 11$$

BBarisan dan Deret Aritmatika

1. Mengenal Barisan Aritmatika

Bayangkan seorang tukang batu yang menyusun batu bata dalam beberapa baris. Baris pertama 5 batu, baris kedua 8 batu, baris ketiga 11 batu, baris keempat 14 batu, dan seterusnya. Setiap baris selalu bertambah 3 batu dari baris sebelumnya.

$$5,\quad 8,\quad 11,\quad 14,\quad 17,\quad \ldots$$

Inilah contoh barisan aritmatika — barisan yang setiap sukunya bertambah (atau berkurang) dengan nilai yang sama dan tetap. Nilai tetap itu disebut beda, dilambangkan $b$.

Perhatikan contoh-contoh berikut:

BarisanBeda ($b$)Jenis
$3,\; 7,\; 11,\; 15,\; 19,\; \ldots$$+4$naik
$20,\; 17,\; 14,\; 11,\; 8,\; \ldots$$-3$turun
$5,\; 5,\; 5,\; 5,\; \ldots$$0$konstan
$2,\; 4,\; 8,\; 16,\; \ldots$tidak tetapbukan aritmatika

Cara mudah mengidentifikasi: Hitung selisih $U_2 - U_1$, $U_3 - U_2$, $U_4 - U_3$. Jika ketiga selisih itu sama, barisan tersebut adalah aritmatika.

2. Penurunan Rumus Suku ke-$n$

Misalkan suku pertama adalah $a$ dan bedanya $b$. Terlihat bahwa pada $U_n$, nilai $b$ ditambahkan sebanyak $(n-1)$ kali, karena dari suku ke-1 ke suku ke-$n$ terdapat $(n-1)$ langkah:

$U_1$$U_2$$U_3$$U_4$$\cdots$$U_n$
$a$$a+b$$a+2b$$a+3b$$\cdots$$a+(n{-}1)b$
• Rumus Suku ke-$n$ Barisan Aritmatika
$$U_n = a + (n-1)\,b$$

di mana $a = U_1$ adalah suku pertama dan $b = U_2 - U_1$ adalah beda.

⚠ Catatan Penting

Untuk mengenali barisan aritmatika, periksa apakah selisih setiap dua suku berurutan sama.
Jika $b > 0$: barisan naik  |  Jika $b < 0$: barisan turun  |  Jika $b = 0$: barisan konstan

3. Jumlah $n$ Suku Pertama Deret Aritmatika

Jumlah $n$ suku pertama deret aritmatika diturunkan sebagai berikut:

$$S_n = a + (a+b) + (a+2b) + \cdots + U_n$$ $$S_n = U_n + (U_n-b) + (U_n-2b) + \cdots + a$$

Menjumlahkan kedua baris: $2S_n = \underbrace{(a+U_n) + (a+U_n) + \cdots + (a+U_n)}_{n \text{ kali}}$

Sehingga $2S_n = n\,(a+U_n)$, dan:

• Rumus Jumlah $n$ Suku Pertama Deret Aritmatika
$$S_n = \frac{n}{2}(a + U_n) \qquad \text{atau} \qquad S_n = \frac{n}{2}\bigl[2a + (n-1)\,b\bigr]$$

4. Suku Tengah Barisan Aritmatika

Jika suatu barisan aritmatika memiliki $n$ suku dengan $n$ ganjil, maka terdapat suku tengah:

• Rumus Suku Tengah
$$U_T = U_{\frac{n+1}{2}} = \frac{a + U_n}{2}$$

Suku tengah adalah rata-rata dari suku pertama dan suku terakhir.

📘 Contoh 4

Diketahui barisan aritmatika $3,\; 7,\; 11,\; 15,\; 19,\; \ldots$

  1. Tentukan suku ke-20!
  2. Tentukan jumlah 15 suku pertama!
Penyelesaian:
Diketahui $a = 3$ dan $b = 7 - 3 = 4$.
  1. $U_{20} = a + (20-1)b = 3 + 19 \cdot 4 = 3 + 76 = 79$
  2. $S_{15} = \dfrac{15}{2}\bigl[2(3) + (15-1)(4)\bigr] = \dfrac{15}{2}\bigl[6 + 56\bigr] = \dfrac{15}{2} \cdot 62 = 465$
📘 Contoh 5

Suatu barisan aritmatika diketahui $U_3 = 12$ dan $U_6 = 27$. Tentukan suku ke-9 dan jumlah 9 suku pertama!

Penyelesaian:
Dari $U_n = a + (n-1)b$: $$U_3 = a + 2b = 12 \quad \cdots (1)$$ $$U_6 = a + 5b = 27 \quad \cdots (2)$$ Dari $(2) - (1)$: $3b = 15$, sehingga $b = 5$. Substitusi ke $(1)$: $a = 2$. $$U_9 = 2 + 8 \cdot 5 = 42$$ $$S_9 = \frac{9}{2}(a + U_9) = \frac{9}{2}(2 + 42) = \frac{9}{2} \cdot 44 = 198$$
📘 Contoh 6

Hitunglah nilai dari $2 + 5 + 8 + 11 + \cdots + 50$!

Penyelesaian:
$a = 2$, $b = 3$, $U_n = 50$. Tentukan $n$: $$50 = 2 + (n-1) \cdot 3 \;\Rightarrow\; 48 = 3(n-1) \;\Rightarrow\; n = 17$$ $$S_{17} = \frac{17}{2}(2 + 50) = \frac{17}{2} \cdot 52 = 442$$
📘 Contoh 7

Diketahui $S_n = 2n^2 + 4n$. Tentukan $U_5$ dan beda barisan tersebut!

Penyelesaian: $$S_5 = 2(25) + 4(5) = 70$$ $$S_4 = 2(16) + 4(4) = 48$$ $$U_5 = S_5 - S_4 = 70 - 48 = 22$$ Untuk beda: $U_1 = S_1 = 6$, $\;U_2 = S_2 - S_1 = 16 - 6 = 10$, maka $b = 10 - 6 = 4$.
📘 Contoh 8

Di antara bilangan 4 dan 28 disisipkan 5 bilangan sehingga membentuk barisan aritmatika. Tentukan kelima bilangan yang disisipkan tersebut!

Penyelesaian:
Setelah penyisipan, barisan memiliki $2 + 5 = 7$ suku, dengan $U_1 = 4$ dan $U_7 = 28$. $$U_7 = a + 6b \;\Rightarrow\; 28 = 4 + 6b \;\Rightarrow\; b = 4$$ Kelima bilangan yang disisipkan: $8,\; 12,\; 16,\; 20,\; 24$.
📘 Contoh 9

Suku ke-8 suatu barisan aritmatika adalah 33 dan bedanya 4. Tentukan suku pertama dan jumlah 10 suku pertamanya!

Penyelesaian: $$U_8 = a + 7b \;\Rightarrow\; 33 = a + 7(4) \;\Rightarrow\; a = 33 - 28 = 5$$ $$S_{10} = \frac{10}{2}\bigl[2(5) + (10-1)(4)\bigr] = 5\bigl[10 + 36\bigr] = 5 \times 46 = 230$$
📘 Contoh 10

Jumlah $n$ suku pertama suatu barisan aritmatika dinyatakan sebagai $S_n = 4n^2 - 2n$. Tentukan suku tengah jika barisan tersebut memiliki 11 suku!

Penyelesaian: $$U_1 = S_1 = 4 - 2 = 2$$ $$U_2 = S_2 - S_1 = (16-4) - 2 = 10$$ $$b = U_2 - U_1 = 10 - 2 = 8$$ Suku tengah ke-6 (posisi tengah dari 11 suku): $$U_6 = a + 5b = 2 + 5(8) = 42$$ Verifikasi: $U_T = \dfrac{a + U_{11}}{2} = \dfrac{2 + (2+10\cdot8)}{2} = \dfrac{2+82}{2} = 42$ ✓
📘 Contoh 11

Diketahui barisan aritmatika dengan suku positif. Jumlah tiga suku pertama adalah 21 dan hasil kali suku pertama dan suku ketiga adalah 45. Tentukan suku ke-10!

Penyelesaian:
Misalkan tiga suku pertama adalah $a-b,\; a,\; a+b$. $$(a-b) + a + (a+b) = 21 \;\Rightarrow\; 3a = 21 \;\Rightarrow\; a = 7$$ $$(a-b)(a+b) = 45 \;\Rightarrow\; a^2 - b^2 = 45$$ $$49 - b^2 = 45 \;\Rightarrow\; b^2 = 4 \;\Rightarrow\; b = \pm 2$$ Karena suku positif dan $a = 7$, ambil $b = 2$ (barisan naik) atau $b = -2$ (barisan turun, tetap positif untuk 10 suku pertama).

Suku ke-10 dari suku pertama $U_1 = a - b$: $$U_{10} = (7-2) + 9(2) = 5 + 18 = 23 \quad \text{(untuk } b = 2\text{)}$$
📘 Contoh 12

Sebuah perusahaan konstruksi membangun jembatan dengan susunan tiang baja. Baris pertama menggunakan 8 tiang, setiap baris berikutnya bertambah 3 tiang. Total anggaran untuk tiang adalah Rp 480.000.000 dan setiap tiang berharga Rp 2.000.000. Jika perusahaan ingin memaksimalkan baris yang dapat dibangun, berapa baris maksimum yang bisa dibuat dan berapa sisa anggarannya?

Penyelesaian:
Total tiang yang dapat dibeli: $\dfrac{480.000.000}{2.000.000} = 240$ tiang.

Jumlah tiang untuk $n$ baris: $S_n = \dfrac{n}{2}[2(8)+(n-1)(3)] = \dfrac{n(3n+13)}{2}$

Syarat: $S_n \leq 240$: $$\frac{n(3n+13)}{2} \leq 240 \;\Rightarrow\; 3n^2 + 13n - 480 \leq 0$$ Dengan rumus kuadrat: $n = \dfrac{-13 \pm \sqrt{169 + 5760}}{6} = \dfrac{-13 \pm \sqrt{5929}}{6} = \dfrac{-13 \pm 77}{6}$

Ambil $n$ positif: $n = \dfrac{64}{6} \approx 10{,}67$, maka $n_{\max} = 10$.

Tiang yang digunakan: $S_{10} = \dfrac{10(30+13)}{2} = \dfrac{10 \times 43}{2} = 215$ tiang.

Sisa tiang: $240 - 215 = 25$ tiang $\;\Rightarrow\;$ sisa anggaran $= 25 \times 2.000.000 = \text{Rp 50.000.000}$

CBarisan dan Deret Geometri

1. Mengenal Barisan Geometri

Bayangkan selembar kertas dilipat menjadi dua bagian. Setelah 1 lipatan ada 2 lapisan, setelah 2 lipatan ada 4 lapisan, setelah 3 lipatan ada 8 lapisan, dan seterusnya.

$$1,\quad 2,\quad 4,\quad 8,\quad 16,\quad 32,\quad \ldots$$

Barisan geometri adalah barisan bilangan yang perbandingan (rasio) antara dua suku yang berurutan selalu tetap (konstan). Perbandingan tetap itu disebut rasio, dilambangkan $r$.

$$r = \frac{U_2}{U_1} = \frac{U_3}{U_2} = \frac{U_4}{U_3} = \cdots$$
BarisanRasio ($r$)Sifat
$2,\; 6,\; 18,\; 54,\; \ldots$$r = 3$membesar
$64,\; 32,\; 16,\; 8,\; \ldots$$r = \tfrac{1}{2}$mengecil
$1,\; -2,\; 4,\; -8,\; \ldots$$r = -2$tanda berganti-ganti
$3,\; 6,\; 9,\; 12,\; \ldots$tidak tetapbukan geometri

Cara mudah mengidentifikasi: Hitung $\dfrac{U_2}{U_1}$, $\dfrac{U_3}{U_2}$, $\dfrac{U_4}{U_3}$. Jika ketiga rasio itu sama, barisan tersebut adalah geometri.

2. Penurunan Rumus Suku ke-$n$

Misalkan suku pertama adalah $a$ dan rasionya $r$. Terlihat bahwa pada $U_n$, nilai $r$ dikalikan sebanyak $(n-1)$ kali. Pangkat $r$ selalu satu kurang dari nomor suku:

$U_1$$U_2$$U_3$$U_4$$\cdots$$U_n$
$a$$ar$$ar^2$$ar^3$$\cdots$$ar^{n-1}$
• Rumus Suku ke-$n$ Barisan Geometri
$$U_n = a \cdot r^{n-1}$$

di mana $a = U_1$ adalah suku pertama dan $r = \dfrac{U_2}{U_1}$ adalah rasio ($r \neq 0$).

⚠ Cara membedakan aritmatika dan geometri

Aritmatika: selisih berurutan sama $\;\Rightarrow\;$ $U_n - U_{n-1} = b$ (tetap)
Geometri: rasio berurutan sama $\;\Rightarrow\;$ $\dfrac{U_n}{U_{n-1}} = r$ (tetap)

Jika $|r| > 1$: barisan divergen (membesar)  |  Jika $|r| < 1$: barisan konvergen (mengecil)

3. Jumlah $n$ Suku Pertama Deret Geometri

Untuk menurunkan rumus $S_n$, perhatikan:

$$S_n = a + ar + ar^2 + \cdots + ar^{n-1}$$ $$r \cdot S_n = ar + ar^2 + \cdots + ar^{n-1} + ar^n$$

Mengurangkan: $S_n - r \cdot S_n = a - ar^n$, sehingga $S_n(1-r) = a(1-r^n)$.

• Rumus Jumlah $n$ Suku Pertama Deret Geometri
$$S_n = \frac{a(r^n - 1)}{r - 1} = \frac{a(1 - r^n)}{1 - r}, \qquad r \neq 1$$

Jika $r = 1$, maka $S_n = na$.

4. Suku Tengah Barisan Geometri

Jika suatu barisan geometri memiliki $n$ suku dengan $n$ ganjil:

• Rumus Suku Tengah Barisan Geometri
$$U_T = \sqrt{a \cdot U_n} = \sqrt{U_1 \cdot U_n}$$

Suku tengah adalah rata-rata geometri dari suku pertama dan suku terakhir.

📘 Contoh 13

Diketahui barisan geometri $2,\; 6,\; 18,\; 54,\; \ldots$

  1. Tentukan suku ke-8!
  2. Tentukan jumlah 6 suku pertama!
Penyelesaian:
Diketahui $a = 2$ dan $r = \dfrac{6}{2} = 3$.
  1. $U_8 = 2 \cdot 3^{8-1} = 2 \cdot 3^7 = 2 \cdot 2187 = 4374$
  2. $S_6 = \dfrac{2(3^6 - 1)}{3 - 1} = \dfrac{2(729-1)}{2} = 728$
📘 Contoh 14

Suatu barisan geometri diketahui $U_3 = 12$ dan $U_6 = 96$. Tentukan nilai $a$, $r$, dan $S_5$!

Penyelesaian: $$U_3 = ar^2 = 12 \quad \cdots (1)$$ $$U_6 = ar^5 = 96 \quad \cdots (2)$$ Bagi $(2)$ dengan $(1)$: $r^3 = \dfrac{96}{12} = 8 \;\Rightarrow\; r = 2$
Dari $(1)$: $a \cdot 4 = 12 \;\Rightarrow\; a = 3$. $$S_5 = \frac{3(2^5 - 1)}{2 - 1} = 3(32-1) = 93$$
📘 Contoh 15

Tiga bilangan membentuk barisan geometri. Jumlah ketiga bilangan adalah 14 dan hasil kalinya adalah 64. Tentukan ketiga bilangan tersebut!

Penyelesaian:
Misalkan ketiga bilangan adalah $\dfrac{a}{r},\; a,\; ar$. $$\frac{a}{r} \cdot a \cdot ar = 64 \;\Rightarrow\; a^3 = 64 \;\Rightarrow\; a = 4$$ $$\frac{4}{r} + 4 + 4r = 14 \;\Rightarrow\; 4 + 4r + 4r^2 = 14r$$ $$4r^2 - 10r + 4 = 0 \;\Rightarrow\; 2r^2 - 5r + 2 = 0 \;\Rightarrow\; (2r - 1)(r - 2) = 0$$ Jadi $r = \dfrac{1}{2}$ atau $r = 2$.

Ketiga bilangan: $8,\; 4,\; 2$ atau $2,\; 4,\; 8$.
📘 Contoh 16

Hitunglah nilai dari $3 + 6 + 12 + \cdots + 384$!

Penyelesaian:
$a = 3$, $r = 2$, $U_n = 384$. Tentukan $n$: $$384 = 3 \cdot 2^{n-1} \;\Rightarrow\; 2^{n-1} = 128 = 2^7 \;\Rightarrow\; n = 8$$ $$S_8 = \frac{3(2^8 - 1)}{2-1} = 3(256 - 1) = 765$$
📘 Contoh 17

Barisan $3,\; 3\sqrt{2},\; 6,\; 6\sqrt{2},\; \ldots$ diteruskan sampai 11 suku. Tentukan suku tengahnya!

Penyelesaian:
$a = 3$, $r = \dfrac{3\sqrt{2}}{3} = \sqrt{2}$. $$U_{11} = 3 \cdot (\sqrt{2})^{10} = 3 \cdot 2^5 = 96$$ $$U_T = \sqrt{a \cdot U_{11}} = \sqrt{3 \cdot 96} = \sqrt{288} = 12\sqrt{2}$$

Barisan geometri diketahui $U_1 = 5$ dan $r = 3$. Tentukan $U_6$ dan $S_5$!

Penyelesaian: $$U_6 = 5 \cdot 3^{6-1} = 5 \cdot 243 = 1215$$ $$S_5 = \frac{5(3^5-1)}{3-1} = \frac{5(243-1)}{2} = \frac{5 \times 242}{2} = 605$$
📘 Contoh 19

Suku kedua barisan geometri adalah 6 dan suku kelima adalah 162. Tentukan $a$, $r$, dan $S_6$!

Penyelesaian: $$U_2 = ar = 6 \quad \cdots(1)$$ $$U_5 = ar^4 = 162 \quad \cdots(2)$$ Bagi (2) dengan (1): $r^3 = \dfrac{162}{6} = 27 \;\Rightarrow\; r = 3$. Dari (1): $a = \dfrac{6}{3} = 2$. $$S_6 = \frac{2(3^6-1)}{3-1} = \frac{2(729-1)}{2} = 728$$
📘 Contoh 20

Diketahui barisan geometri dengan semua suku positif. Jumlah suku ke-3 dan ke-5 adalah 180, sedangkan hasil kali suku ke-2 dan ke-6 adalah 2304. Tentukan suku ke-4!

DDeret Geometri Tak Hingga

1. Mengenal Deret Geometri Tak Hingga

Bayangkan kamu memegang satu potong batang kayu. Kamu memotongnya menjadi dua bagian sama besar, lalu mengambil satu bagian ($\tfrac{1}{2}$ batang). Dari sisa batang kayu yang ada, kamu potong lagi menjadi dua dan ambil satu bagian lagi ($\tfrac{1}{4}$ batang). Begitu seterusnya tanpa henti:

$$\frac{1}{2} + \frac{1}{4} + \frac{1}{8} + \frac{1}{16} + \cdots$$

Meskipun proses memotong dilakukan tak terbatas kali, total batang kayu yang kamu ambil tidak pernah melebihi 1 batang kayu utuh — bahkan mendekati tepat 1 batang kayu. Inilah contoh nyata deret geometri tak hingga yang konvergen.

Sebaliknya, bayangkan potongan batang kayu yang justru bertambah besar setiap kali — tentu saja batang kayu yang dibutuhkan akan terus bertambah tanpa batas. Inilah gambaran deret yang divergen.

Kunci perbedaannya ada pada rasio $r$:

  • Jika $|r| < 1$: setiap potongan semakin kecil mendekati nol $\;\Rightarrow\;$ deret konvergen
  • Jika $|r| \geq 1$: potongan tidak mengecil $\;\Rightarrow\;$ deret divergen

2. Penurunan Rumus Deret Geometri Tak Hingga

Dari rumus $S_n = \dfrac{a(1-r^n)}{1-r}$ (untuk $r \neq 1$), kita amati apa yang terjadi ketika $n \to \infty$:

  • Jika $|r| < 1$, maka $r^n \to 0$ saat $n \to \infty$
  • Sehingga $S_n \to \dfrac{a(1-0)}{1-r} = \dfrac{a}{1-r}$
• Rumus Deret Geometri Tak Hingga (Konvergen)
$$S_\infty = \frac{a}{1-r}, \qquad \text{syarat: } |r| < 1 \;\; (-1 < r < 1)$$

Jika $|r| \geq 1$, deret divergen dan jumlahnya tidak berhingga.

⚠ Catatan Penting

Syarat konvergensi: $-1 < r < 1$ atau $|r| < 1$.

Deret geometri tak hingga sering muncul pada masalah lintasan bola memantul dan pecahan desimal berulang. Rumus ini hanya berlaku bila syarat di atas terpenuhi — selalu periksa nilai $r$ terlebih dahulu!

EAplikasi Barisan dan Deret

Barisan dan deret bukan sekadar kumpulan angka di atas kertas — keduanya hadir nyata dalam kehidupan sehari-hari dan menjadi alat penting untuk memahami serta memecahkan berbagai permasalahan di dunia nyata. Mulai dari menghitung cicilan rumah, memprediksi pertumbuhan penduduk, hingga menganalisis peluruhan zat radioaktif, semuanya menggunakan konsep yang telah kita pelajari.

Secara umum, pilihan jenis barisan dalam pemodelan mengikuti pola berikut:

  • Gunakan barisan aritmatika jika perubahan bersifat tetap/konstan (bertambah atau berkurang dengan jumlah yang sama setiap periode).
  • Gunakan barisan geometri jika perubahan bersifat persentase/proporsional (bertambah atau berkurang dengan faktor pengali yang sama setiap periode).
  • Gunakan deret geometri tak hingga jika proses terus berulang dengan rasio $|r| < 1$ hingga nilai mendekati nol.

1. Bunga Tunggal dan Bunga Majemuk

• Bunga Tunggal (pola aritmatika)
$$M_n = M_0(1 + i \cdot n)$$

$M_n$ = besar modal setelah $n$ periode  |  $M_0$ = modal awal  |  $i$ = suku bunga per periode  |  $n$ = banyaknya periode

• Bunga Majemuk (pola geometri)
$$M_n = M_0(1 + i)^n$$

$M_n$ = besar modal setelah $n$ periode  |  $M_0$ = modal awal  |  $i$ = suku bunga per periode  |  $n$ = banyaknya periode

⚠ Catatan Penting

Bunga Tunggal: Bunga dihitung secara tetap berdasarkan modal awal ($M_0$), sehingga besar bunga tidak berubah di setiap periodenya.

Bunga Majemuk: Bunga dihitung berdasarkan modal yang telah terakumulasi dengan bunga pada periode sebelumnya (bunga berbunga).

📘 Contoh 29

Penyelesaian: $$M_8 = 5.000.000\,(1 + 0{,}03 \times 8) = 5.000.000 \times 1{,}24 = \text{Rp 6.200.000}$$

2. Pertumbuhan dan Peluruhan

• Pertumbuhan Aritmatika
$$M_n = M_0 + b \cdot n$$

$M_n$ = nilai setelah $n$ periode  |  $M_0$ = nilai awal  |  $b$ = pertambahan tetap setiap periode  |  $n$ = banyaknya periode

• Pertumbuhan Geometri
$$M_n = M_0 \cdot r^n \quad (r > 1) \qquad \text{atau} \qquad M_n = M_0(1+p)^n$$

$r$ = rasio (faktor pengali tetap)  |  $p$ = persentase pertumbuhan per periode (desimal), sehingga $r = 1 + p$

• Peluruhan Aritmatika
$$M_n = M_0 - b \cdot n$$

$b$ = pengurangan tetap setiap periode

• Peluruhan Geometri
$$M_n = M_0 \cdot r^n \quad (0 < r < 1) \qquad \text{atau} \qquad M_n = M_0(1-p)^n$$

$r$ = rasio, $0 < r < 1$ untuk peluruhan  |  $p$ = persentase penurunan per periode, sehingga $r = 1 - p$

FPengayaan: Bentuk Linier $U_n$, Bentuk Kuadrat $S_n$, Beda, dan Rasio

Bagian pengayaan ini membahas tiga topik lanjutan: (1) menurunkan rumus $U_n$ ke bentuk linier dan $S_n$ ke bentuk kuadrat langsung dari definisi, (2) sifat selisih suku (rahasia beda), dan (3) sifat perbandingan suku (rahasia rasio).

1. Penurunan Bentuk $U_n = bn + (a - b)$ dan $S_n = \dfrac{b}{2}n^2 + \left(a - \dfrac{b}{2}\right)n$

Kita mulai dari rumus $U_n = a + (n-1)b$ dan manipulasi aljabarnya:

$$U_n = a + (n-1)b = a + bn - b = bn + a - b$$
• Bentuk Linier Suku ke-$n$ Barisan Aritmatika
$$U_n = bn + (a - b)$$

di mana koefisien $n$ adalah beda $b$, dan konstanta adalah $(a - b)$.

Cara membaca $a$ dan $b$ dari rumus $U_n$:
Jika diketahui $U_n = pn + q$ (bentuk linier dalam $n$), maka: $$b = p \qquad \text{dan} \qquad a - b = q \;\Rightarrow\; a = p + q$$ Contoh cepat: $U_n = 3n + 4 \;\Rightarrow\; b = 3$ dan $a = 3 + 4 = 7$, sehingga barisan: $7,\; 10,\; 13,\; \ldots$

Dari rumus $S_n = \dfrac{n}{2}\bigl[2a + (n-1)b\bigr]$ dan dikembangkan:

$$S_n = \frac{b}{2}n^2 + \frac{2a - b}{2}\,n = \frac{b}{2}n^2 + \left(a - \frac{b}{2}\right)n$$
• Bentuk Kuadrat Jumlah $n$ Suku Pertama Deret Aritmatika
$$S_n = \frac{b}{2}n^2 + \left(a - \frac{b}{2}\right)n$$

di mana koefisien $n^2$ adalah $\dfrac{b}{2}$, dan koefisien $n$ adalah $\left(a - \dfrac{b}{2}\right)$.

Cara membaca $a$ dan $b$ dari rumus $S_n$:
Jika diketahui $S_n = \alpha n^2 + \beta n$ (bentuk kuadrat dalam $n$), dengan membandingkan: $$\frac{b}{2} = \alpha \;\Rightarrow\; b = 2\alpha \qquad \text{dan} \qquad a - \frac{b}{2} = \beta \;\Rightarrow\; a = \alpha + \beta$$

⚠ Ringkasan membaca koefisien

$$S_n = \underbrace{\alpha}_{\frac{b}{2}} n^2 + \underbrace{\beta}_{a\,-\,\frac{b}{2}} n \quad\Longrightarrow\quad b = 2\alpha, \quad a = \alpha + \beta$$ $$U_n = \underbrace{p}_{b}\, n + \underbrace{q}_{a\,-\,b} \quad\Longrightarrow\quad b = p, \quad a = p + q$$

📘 Contoh 37

Diketahui barisan $1,\; 3,\; 5,\; \ldots$ dengan $a = 1$ dan $b = 2$. Turunkan rumus $U_n$ ke bentuk linier dan $S_n$ ke bentuk kuadrat, kemudian verifikasi dengan $n = 3$!

Penyelesaian:
Rumus $U_n$ bentuk linier: $$U_n = bn + (a - b) = 2n + (1 - 2) = 2n - 1$$ Rumus $S_n$ bentuk kuadrat: $$S_n = \frac{b}{2}n^2 + \left(a - \frac{b}{2}\right)n = \frac{2}{2}n^2 + \left(1 - \frac{2}{2}\right)n = n^2 + 0 \cdot n = n^2$$ Verifikasi $n = 3$: $U_3 = 2(3)-1 = 5$ ✓   dan   $S_3 = 3^2 = 9 = 1+3+5$ ✓
📘 Contoh 38

Diketahui $S_n = 4n^2 + 5n$. Tentukan nilai $b$, $a$, dan rumus $U_n$!

Penyelesaian:
Dari $S_n = 4n^2 + 5n$: $\alpha = 4$, $\beta = 5$. $$b = 2\alpha = 2 \times 4 = 8$$ $$a = \alpha + \beta = 4 + 5 = 9$$ Rumus $U_n$ bentuk linier: $$U_n = bn + (a - b) = 8n + (9 - 8) = 8n + 1$$ Verifikasi: $U_1 = 8(1)+1 = 9 = a$ ✓   $U_2 - U_1 = 17 - 9 = 8 = b$ ✓
📘 Contoh 39

Diketahui $U_n = 4n - 5$. Tentukan $b$, $a$, dan rumus $S_n$ bentuk kuadrat!

Penyelesaian:
Dari $U_n = 4n - 5$: $p = 4$, $q = -5$. $$b = p = 4$$ $$a = p + q = 4 + (-5) = -1$$ Rumus $S_n$ bentuk kuadrat: $$S_n = \frac{b}{2}n^2 + \left(a - \frac{b}{2}\right)n = \frac{4}{2}n^2 + \left(-1 - \frac{4}{2}\right)n = 2n^2 - 3n$$ Verifikasi: $S_3 = 2(9) - 3(3) = 18 - 9 = 9 = (-1) + 3 + 7 = 9$ ✓
📘 Contoh 40

Diketahui $S_n = 2n^2 - 3n$. Tentukan $b$, $a$, rumus $U_n$, dan nilai $U_4$!

Penyelesaian:
$\alpha = 2$, $\beta = -3$. $$b = 2(2) = 4$$ $$a = 2 + (-3) = -1$$ $$U_n = 4n + (-1 - 4) = 4n - 5$$ $$U_4 = 4(4) - 5 = 11$$

2. Rahasia Beda: Selisih Suku pada Barisan Aritmatika

Bentuk linier $U_n = bn + (a-b)$ memperlihatkan bahwa $U_n$ adalah fungsi linear terhadap $n$. Dari sini kita dapat menurunkan sifat selisih antar suku secara langsung:

$$U_n - U_k = [bn + (a-b)] - [bk + (a-b)] = bn - bk = b(n-k)$$
• Sifat Selisih Suku Barisan Aritmatika (Rahasia Beda)
$$U_n - U_k = (n - k)\,b$$

Selisih dua suku mana pun sama dengan selisih nomor sukunya dikali beda.

⚠ Catatan

Sifat ini memungkinkan kita menentukan beda $b$ secara langsung dari dua suku yang diketahui tanpa perlu mencari suku pertama terlebih dahulu: $$b = \frac{U_n - U_k}{n - k}$$

📘 Contoh 41

Diketahui barisan aritmatika $2,\; 4,\; 6,\; 8,\; \ldots$ ($b = 2$). Verifikasikan bahwa $U_3 - U_1 = 2b$ dan $U_4 - U_1 = 3b$!

Penyelesaian: $$U_3 - U_1 = 6 - 2 = 4 = (3-1) \times 2 = 2b \quad \checkmark$$ $$U_4 - U_1 = 8 - 2 = 6 = (4-1) \times 2 = 3b \quad \checkmark$$
📘 Contoh 42

Barisan aritmatika diketahui $U_3 = 10$ dan $U_7 = 26$. Tentukan $b$, $a$, dan $U_{10}$!

Penyelesaian:
Menggunakan rahasia beda: $$U_7 - U_3 = (7-3)\,b \;\Rightarrow\; 26 - 10 = 4b \;\Rightarrow\; b = 4$$ $$a = U_3 - (3-1)b = 10 - 2(4) = 10 - 8 = 2$$ $$U_{10} = 4(10) + (2-4) = 40 - 2 = 38$$

3. Rahasia Rasio: Perbandingan Suku pada Barisan Geometri

Pada barisan geometri, analog dengan rahasia beda pada aritmatika, terdapat sifat perbandingan antar suku yang diturunkan langsung dari $U_n = ar^{n-1}$:

$$\frac{U_n}{U_k} = \frac{a \cdot r^{n-1}}{a \cdot r^{k-1}} = r^{(n-1)-(k-1)} = r^{n-k}$$
• Sifat Perbandingan Suku Barisan Geometri (Rahasia Rasio)
$$\frac{U_n}{U_k} = r^{n-k}$$

Perbandingan dua suku mana pun sama dengan rasio dipangkatkan selisih nomor sukunya.

⚠ Catatan

Sifat ini memungkinkan kita menentukan rasio $r$ secara langsung tanpa mencari suku pertama: $$r^{n-k} = \frac{U_n}{U_k} \;\Rightarrow\; r = \left(\frac{U_n}{U_k}\right)^{\!\frac{1}{n-k}}$$

📘 Contoh 43

Diketahui barisan geometri $2,\; 4,\; 8,\; 16,\; \ldots$ ($r = 2$). Verifikasikan bahwa $\dfrac{U_3}{U_1} = r^2$ dan $\dfrac{U_4}{U_1} = r^3$!

Penyelesaian: $$\frac{U_3}{U_1} = \frac{8}{2} = 4 = 2^{3-1} = r^2 \quad \checkmark$$ $$\frac{U_4}{U_1} = \frac{16}{2} = 8 = 2^{4-1} = r^3 \quad \checkmark$$
📘 Contoh 44

Barisan geometri diketahui $U_2 = 4$ dan $U_4 = 16$. Tentukan $r$, $a$, dan $U_6$!

Penyelesaian:
Menggunakan rahasia rasio: $$\frac{U_4}{U_2} = r^{4-2} = r^2 \;\Rightarrow\; \frac{16}{4} = r^2 \;\Rightarrow\; r = 2$$ $$a = \frac{U_2}{r} = \frac{4}{2} = 2$$ $$U_6 = U_4 \cdot r^{6-4} = 16 \times 2^2 = 16 \times 4 = 64$$
