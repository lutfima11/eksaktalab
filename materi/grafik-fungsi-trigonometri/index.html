AGrafik Fungsi Trigonometri Dasar

Grafik fungsi trigonometri digambarkan pada koordinat Cartesius dengan sumbu-$X$ sebagai nilai sudut dan sumbu-$Y$ sebagai nilai fungsi. Tiga komponen penting: nilai maksimum, nilai minimum, dan periode.

(a) Grafik $y = \sin x$

• Grafik y = sin x

(b) Grafik $y = \cos x$

• Grafik y = cos x

(c) Grafik $y = \tan x$

• Grafik y = tan x

$y = \cos x$$1$$-1$$360^\circ$
$y = \tan x$tidak terbatas (unbounded)$180^\circ$

BFungsi Trigonometri Sederhana

• Bentuk Umum Fungsi Trigonometri Sederhana
$$y = k\sin(ax \pm \alpha) \qquad y = k\cos(ax \pm \alpha) \qquad y = k\tan(ax \pm \alpha)$$
  • Nilai maksimum $= k$ dan nilai minimum $= -k$
  • Periode $= \dfrac{360^\circ}{a}$ (sin & cos) atau $\dfrac{180^\circ}{a}$ (tan)
  • Jika $+\alpha$: grafik geser kiri; jika $-\alpha$: grafik geser kanan
⚠ Catatan untuk Bentuk $y = p + k\sin(ax)$

Untuk fungsi bentuk $y = p + k\sin(ax)$, nilai maks $= p + k$ dan min $= p - k$.

📘 Contoh 1

Tentukan amplitudo, nilai maksimum, nilai minimum, dan periode dari:

  1. $y = 3\sin x$
  2. $y = -2\cos x$
  3. $y = 1 + 4\sin 2x$
  4. $y = -3 + 2\cos 3x$
  1. Amplitudo $= 3$, maks $= 3$, min $= -3$, periode $= 360^\circ$.
  2. Amplitudo $= 2$. Tanda negatif membalik grafik: maks $= 2$, min $= -2$, periode $= 360^\circ$.
  3. Amplitudo $= 4$. Maks $= 1 + 4 = 5$, min $= 1 - 4 = -3$, periode $= \dfrac{360^\circ}{2} = 180^\circ$.
  4. Amplitudo $= 2$. Karena ada tanda $-$ di depan, grafik terbalik. Maks $= -3 + 2 = -1$, min $= -3 - 2 = -5$, periode $= \dfrac{360^\circ}{3} = 120^\circ$.
📘 Contoh 2

Tentukan nilai maksimum, minimum, dan periode dari:

  1. $y = 5\sin(3x - 60^\circ)$
  2. $y = 3\cos(2x + 45^\circ)$
  3. $y = 6\tan 2x$
  4. $y = 4 + 2\cos 5x$
  5. $y = 5 - 3\cos\!\left(\tfrac{1}{3}x + \pi\right)$
FungsiMaksMinPeriode
(a)$y = 5\sin(3x - 60^\circ)$$5$$-5$$120^\circ$
(b)$y = 3\cos(2x + 45^\circ)$$3$$-3$$180^\circ$
(c)$y = 6\tan 2x$tidak terbatas$90^\circ$
(d)$y = 4 + 2\cos 5x$$6$$2$$72^\circ$
(e)$y = 5 - 3\cos\!\left(\tfrac{1}{3}x + \pi\right)$$8$$2$$1080^\circ$

Untuk (e): tanda $-$ di depan cosinus membalik. Maks saat $\cos = -1$: $5-3(-1) = 8$; min saat $\cos = 1$: $5-3(1) = 2$.

📘 Contoh 3

Tentukan nilai maksimum dan minimum dari $f(x) = \dfrac{24}{5 + 3\sin x}$!

Maks terjadi saat penyebut terkecil ($\sin x = -1$): $f_{\max} = \dfrac{24}{5-3} = \dfrac{24}{2} = \mathbf{12}$

Min terjadi saat penyebut terbesar ($\sin x = 1$): $f_{\min} = \dfrac{24}{5+3} = \dfrac{24}{8} = \mathbf{3}$

📘 Contoh 4

Tentukan nilai maksimum dan minimum dari:

  1. $f(x) = \dfrac{3}{2 - \sin x}$
  2. $g(x) = \dfrac{8}{3 + 5\cos x}$
  1. Maks saat $\sin x = 1$: $f_{\max} = \dfrac{3}{2-1} = 3$. Min saat $\sin x = -1$: $f_{\min} = \dfrac{3}{2+1} = 1$.
  2. Penyebut $3 + 5\cos x$ berada antara $-2$ dan $8$. Penyebut bisa negatif! Min (paling negatif) saat penyebut $= -2$: $g_{\min} = \dfrac{8}{-2} = -4$. Maks saat penyebut $= 8$: $g_{\max} = \dfrac{8}{8} = 1$.
⚠ Perhatian Fungsi Pecah

Fungsi bentuk $\dfrac{k}{a \pm b\sin x}$ harus diperiksa tanda penyebutnya. Jika penyebut bisa berganti tanda, fungsi tidak terdefinisi di beberapa titik dan perlu dianalisis lebih hati-hati.

CMenentukan Persamaan Fungsi dari Grafik

• Langkah Menentukan Persamaan dari Grafik
  1. Tentukan amplitudo: $k = \dfrac{\text{maks} - \text{min}}{2}$
  2. Tentukan periode dari grafik, lalu hitung $a = \dfrac{360^\circ}{\text{periode}}$
  3. Tentukan pergeseran vertikal: $c = \dfrac{\text{maks} + \text{min}}{2}$
  4. Tentukan pergeseran fase $\alpha$ dari posisi maksimum/titik nol naik
  5. Tuliskan persamaan: $y = k\sin a(x - \alpha) + c$ atau $y = k\cos a(x - \alpha) + c$
📘 Contoh 5

Lukislah grafik $f(x) = 2\cos x$ pada interval $0^\circ < x \leq 360^\circ$!

Amplitudo $= 2$, periode $= 360^\circ$. Titik kunci: $(0^\circ, 2)$, $(90^\circ, 0)$, $(180^\circ, -2)$, $(270^\circ, 0)$, $(360^\circ, 2)$.

📘 Contoh 6

Lukislah grafik $f(x) = \sin(x - 45^\circ)$ pada interval $0^\circ < x \leq 360^\circ$!

Grafik $y = \sin x$ digeser ke kanan sejauh $45^\circ$. Titik kunci bergeser: nol naik di $45^\circ$, puncak di $135^\circ$, nol turun di $225^\circ$, lembah di $315^\circ$.

📘 Contoh 7

Tentukan periode dan letak asimtot vertikal dari $f(x) = \tan\!\left(\dfrac{1}{2}x\right)$ pada interval $[0^\circ, 360^\circ]$!

$a = \dfrac{1}{2}$, periode $= \dfrac{180^\circ}{1/2} = 360^\circ$.

Asimtot saat $\dfrac{1}{2}x = 90^\circ \implies x = 180^\circ$. Dalam $[0^\circ, 360^\circ]$, asimtot di $x = 180^\circ$.

📘 Contoh 8

Sebuah grafik fungsi sinus memiliki nilai maks $= 3$, nilai min $= -3$, titik nol naik pertama di $x = 75^\circ$, dan berikutnya di $x = 255^\circ$. Tentukan persamaan fungsinya!

Langkah 1: Amplitudo $k = 3$.

Langkah 2: Periode $= 255^\circ - 75^\circ = 180^\circ$, $a = \dfrac{360^\circ}{180^\circ} = 2$.

Langkah 3: Nol naik di $75^\circ$ (geser kanan $75^\circ$):

$$\boxed{y = 3\sin\bigl(2(x-75^\circ)\bigr) = 3\sin(2x - 150^\circ)}$$
📘 Contoh 9

Sebuah grafik fungsi trigonometri memiliki nilai maks $= 3$ di $x = 60^\circ$, nilai min $= -1$ di $x = 240^\circ$, dan garis tengah $y = 1$. Tentukan persamaan fungsinya dalam bentuk $y = k\cos(a(x - \alpha)) + c$!

Langkah 1: $k = \dfrac{3-(-1)}{2} = 2$, $c = \dfrac{3+(-1)}{2} = 1$.

Langkah 2: $\dfrac{T}{2} = 240^\circ - 60^\circ = 180^\circ \implies T = 360^\circ \implies a = 1$.

Langkah 3: Maks cosinus standar di $x = 0^\circ$, grafik ini di $x = 60^\circ$ → geser kanan $60^\circ$.

$$\boxed{y = 2\cos(x - 60^\circ) + 1}$$
📘 Contoh 10

Sebuah grafik fungsi trigonometri memiliki: maks $= 4$, min $= -2$, periode $= 240^\circ$, dan melewati titik $(30^\circ,\, 1)$. Tentukan persamaannya dalam bentuk $y = k\sin(ax + b) + c$!

Langkah 1: $k = \dfrac{4-(-2)}{2} = 3$, $c = \dfrac{4+(-2)}{2} = 1$.

Langkah 2: $a = \dfrac{360^\circ}{240^\circ} = \dfrac{3}{2}$.

Langkah 3: Substitusi $(30^\circ, 1)$ ke $1 = 3\sin\!\left(\dfrac{3}{2} \cdot 30^\circ + b\right) + 1$:

$$0 = 3\sin(45^\circ + b) \implies 45^\circ + b = 0^\circ \implies b = -45^\circ$$
$$\boxed{y = 3\sin\!\left(\frac{3}{2}x - 45^\circ\right) + 1}$$

  2. Lukislah grafik $f(x) = 3\sin 2x$ pada interval $0^\circ \leq x \leq 360^\circ$ dengan membuat tabel nilai terlebih dahulu!
  3. Lukislah grafik $f(x) = 2\cos(x + 30^\circ)$ pada interval $0^\circ \leq x \leq 360^\circ$ dan bandingkan dengan $y = \cos x$! Jelaskan pengaruh koefisien 2 dan pergeseran $30^\circ$.
  4. Lukislah grafik $f(x) = 2\tan\!\left(\dfrac{1}{3}x\right)$ pada $0^\circ \leq x \leq 360^\circ$. Tentukan: (a) periode, (b) letak asimtot vertikal, (c) perbedaan dengan $y = \tan x$.
  5. Tentukan persamaan fungsi sinus dengan amplitudo 4, periode $120^\circ$, dan pergeseran ke kanan sejauh $20^\circ$. Kemudian tentukan nilai fungsi tersebut pada $x = 50^\circ$!
  6. Sebuah grafik fungsi trigonometri memiliki nilai maks $= 5$, min $= -5$, titik nol naik pertama di $x = 30^\circ$, dan titik nol naik berikutnya di $x = 210^\circ$. (a) Tentukan amplitudo, periode, dan nilai $a$. (b) Tentukan persamaan fungsi dalam bentuk $y = k\sin(ax + b)$. (c) Verifikasi dengan mensubstitusi $x = 30^\circ$.
